## Язык программирования
***
Алгоритмический язык программирования, синтаксис совпадает с синтаксисом С. Каждая строчка должна заканчиваться
знаком `;`. 

Статическая типизация, существует 3 типа данных:
* `int` - целое число
* `char` - символ - целое число от 0 до 128
* `char[size]` - массив символов, строка
